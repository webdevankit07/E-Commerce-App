## Run Locally

Clone the project

```bash
  git clone https://github.com/webdevankit07/E-Commerce-App.git
```

Go to the project directory

```bash
  cd E-Commerce-App
```

Install Frontend dependencies

```bash
  npm install
```

Start the Frontend server

```bash
  npm run dev
```
